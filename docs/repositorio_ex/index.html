Como utilizar o Repositório de Exercícios?

Preparamos um repositório extra que contém atividades para cada notebook presente no repositório de capacitação. O objetivo é que você possa praticar seus conhecimentos, e servir realmente como um playground. Não pretendemos ativamente corrigir as atividades (a não ser em caso de dúvidas), apenas verificar o andamento delas e como vocês estão se virando. Portanto, aproveitem o material!

Como utilizar este material

Acesse a página com as instruções de instalação completas clicando aqui.

Como configurar o repositório de exercícios?

  1. Acesse o repositório de exercícios: clique aqui.
  2. Na parte superior direita, clique em Fork e defina um nome para o seu repositório. Deixe-o público para ser visível para os outros membros e diretores.
  3. Clone seu repositório localmente:
    • Abra o terminal e digite git clone (link para repositório).
    • Navegue para o repositório utilizando cd nome_do_repo.
  4. Adicione o repositório de exercícios como um repositório upstream remoto: git remote add upstream https://github.com/thomaschiari/ML_AI-Training-Submits.git.
  5. Abra seu repositório na sua IDE de preferência e realize as atividades.
  6. Para atualizar as suas edições no repositório remoto, utilize os seguintes comandos:
    • git add arquivos_a_adicionar
    • git commit -m "mensagem_de_commit"
    • git push origin main
  7. Para manter seu repositório atualizado e seguindo o repositório de origem, utilize os seguintes comandos:
    • git fetch upstream
    • git merge upstream/main
    • git push origin main
  8. Pronto! Agora seu repositório está configurado e atualizado.
